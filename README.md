# ReallySimpleCRM

a really simple crm.

## Setup

Prerequisites:
* python3
* virtualenv

1. Activate the virtual environment.
```
source rscrm-venv/bin/activate
```

2. Start the server.
```
python3 crm/manage.py runserver
```

3. Navigate to the provided address.
